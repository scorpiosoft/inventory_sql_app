# inventory_sql_app
Present a storefront to a SQL inventory DB

## Built With

Node.js

Node APIs:
* mysql
* inquirer
* sprintf-js

### Prerequisites

- [Node.js](https://nodejs.org/en/)
- [npm](https://www.npmjs.com)

### Installing

1. Clone this repo down to your local machine
2. `npm install`

### Running

1. `node customer`
2. `node manager`
